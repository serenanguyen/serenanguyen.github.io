Greenbelt Now

Greenbelt Now is responsive React web app that displays water data from the USGS API using React Hooks for state management. Users can select locations along the Greenbelt from an interactive map that I created in Adobe Illustrator.

Southwire Configurator

Southwire Configurator is an interactive web app using technologies such as React, Redux, and Firestore with responsive design supporting desktop, tablets, and mobile devices. I was the lead developer on this project working with designers, QA engineers, and project managers to clarify requirements and manage dependencies. I implemented features that rendered downloadable PDFs from XLS files uploaded by users. I also provided production support to improve product usability after user feedback.

Jr. NBA Coach

Jr. NBA Coach is a React Native app using MobX state management. I worked with designers to build screens and components that satisfied both iOS and android UI/UX standards. I also maintained dependencies to meet Google Play and App Store requirements.
